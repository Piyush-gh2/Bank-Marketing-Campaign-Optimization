# 🏦 Bank Marketing AIOps Project

## 📌 Overview
An AI-driven AIOps framework designed to optimize bank marketing campaigns by predicting customer conversion probability, detecting anomalies, and automating alerts.

## 🚀 Features
- Customer Conversion Prediction (Random Forest)
- Campaign Anomaly Detection (Isolation Forest)
- Automated Alert Engine
- Streamlit Dashboard
- End-to-End ML Pipeline

## 🛠 Tech Stack
- Python
- Scikit-learn
- Pandas
- Streamlit
- Git

## 📊 Architecture
Bank Data → Preprocessing → ML Model → Anomaly Detection → Alert Engine → Dashboard

## ▶ How to Run
python main.py
streamlit run dashboard/app.py
